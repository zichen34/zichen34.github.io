Saliency in images

unique features 1

Saliency map

A scalar quantity representing the conspicuity to guide the selection of attended locations, based on the spatial distribution of saliency. 2

Steps:

  1. Process input image: colors, intensity and orientations
    • Color: the input image is convert to red-green-blue-yellow color space.
    • Intensity: the input image is converted to grayscale.
    • Orientation feature: the input image are processed by Gabor filters with respect to 4 angles
  2. Create Gaussian pyramids for each of the 3 processed images
  3. Saliency map is the mean of 3 Gaussian pyramids

Another Saliency Map ?

To interpret how a deep learning model achieve classification. One way is to see the amount of contribution to the predicted class score for each pixel .

Since pixels are combined together with different weights (𝐖ᵀ𝐈+𝐛) to form ultimately another representation to perform classification, the weight of the pixel indicates its importance to the class label.5

The gradient of the objective function (correct label score Sc=𝐖ᵀ𝐈+𝐛) with respect to a pixel: w=∂Sc / ∂I
