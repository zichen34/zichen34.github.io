fov

Code from 3DGS

  • Field of view: fovX = $2* arctan(\frac{width}{2f})$
  • Near plane’s right boundary: $z_{near} * tan(fovX)$

  1. Convert fov to focal

    1
2

    def fov2focal(fovX, width):   # 1111.11103, 800
    return width / (2 * math.tan(fov / 2))
