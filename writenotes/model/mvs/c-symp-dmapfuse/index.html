    (2024-03-16)

    • 4.3 Fusion

      1. Obtain depth map for each view
      2. Iterate each depth map to be treated as the reference image and perform reprojection.
      3. Check if the disparity between the reference and source depth maps is lower than a threshold, so as to determine valid 3D points.
      4. Check if the estimated surface normal direction consistent among all views.
      5. Check the if reprojected depth on other view is consistent with the reference view.

    • Accuracy and completeness is a trade-off.

    (2024-03-15)

    • Goal of depth map fusion is recovering the surface.

    • Use coordinate descent to decompose the energy functional (泛函) for each voxel.

    • 3D total variance TV is the loss function. So the Objective function = TV + regularizer L. Naming refers to Fast Optimization Methods for L1 Regularization - ECML07 and MIT courseware

    • Scene Representation: TSDF for surface reconstruction.

    • Depth maps are generated based on binocular stereo matching using library libelas.

    • Leverage TSDFs to fuse depth maps

    • Eq (1) is an expection of all-view TSDFs on a voxel. It fails for nosiy depth maps due to lack of spatial regularizer.

      $$ min \ \int_B ∑_{i∈ Φ(𝐱)} w_i (u - f_i)² d𝐱 $$

      • Integral over all voxels in the bounding box $B$
      • $fᵢ$ is a TSDF set generated from the i-th depth map.
      • Each TSDF corresponds a weight $wᵢ$ representing its reliability.
      • $Φ$ is the filtered TSDF set for a voxel $𝐱=B(i,j,k)$. And the original set is $F$ that contains TSDFs on 𝐱 from all depth maps.
      • u is the resultant TSDF. It will finally be the weighted mean of depth values on all dpeth maps.
