Extreme Learning Machine With Subnetwork Hidden Nodes for Regression and Classification (Cybernet 2015)

8 minute read

IEEE Cybernetics(2015-11-02) | Google Drive | G.Scholar

  • This is the first paper of ELM with subnetwork nodes by Yimin Yang.
    • The second paper in the series is MltLyr ELM with subnetwork nodes
  • The outline of Yang’s research works: Yang-WeChatLive-20221020;

Abstract

  • Learning effectiveness and speed of SLFN are bottleneck.
  • ELM is fast.
  • Grow subnetwork nodes by pulling back residual network error to the hidden layer.
  • Better generalization performance with fewr hidden nodes.

в… . Introduction

  • Bring out the subject:
    FNN (universial approximator) вһ” SLFN

  • What is an SLFN?
    Input layer + hidden layer + output layer

  • Math description:
    For N arbitary distinct samples {(рқҗұбөў,рқҗӯбөў)}бөўвӮҢвӮҒбҙә, where рқҗұбөўвҲҲ рқҗ‘вҒҝ and рқҗӯбөўвҲҲ рқҗ‘бөҗ, the network output is:

    $рқҗҹ_L(рқҗұ)$ = вҲ‘бөўвӮҢвӮҒбҙё рқӣғбөўh(рқҗҡбөўвӢ…рқҗұвұј + bбөў) = вҲ‘бөўвӮҢвӮҒбҙё рқҗҮбөўвӢ…рқӣғбөў, j=1,…,N вҖғ (1)

    • SLFN output is the weighted sum of рқҗҝ hidden nodes (perceptrons) with the factor рқӣғ.
    • The ith perceptron receives the weighted sum of рқ‘Ғ inputs through its parameters (рқҗҡбөў, bбөў), рқҗҡбөўвҲҲ рқҗ‘вҒҝ, bвҲҲ рқҗ‘, and performs activation function h. Its contribution ratio to the all output nodes is рқӣғбөў.

  • ELM traits:
    NN (all params are adjustable) вһ” partial random networks вһ” ELM is a full-random learning method, where the input weights and bias (рқҗҡ, b) are generated randomly and independent of training data. (Will the Glorot normalization has no effect?)

  • ELM advantages:
    An unification of FNNs and SVM/LS-SVM

  • ELM application:
    CV, da,…, online learning

  • Problems:

    1. The choice of the regularization parameter C which affects the generalization performance of ELM mainly relies on trial-and-error method.

    2. How many neurons should be used in ELM. Although Huang suggested to use more than 1000 hidden nodes, whether the number of hidden nodes can be further reduced without affecting learning effectiveness for large-size/high dimension data set.
      Several improved ELM methods, like B-ELM pulls the network residual error back to the hidden layer but it only works for regression task, and other methods bring a higher computation complexity when compared to standard ELM.

  • Solution: Growing subnetwork hidden nodes to the exisiting network by pulling back the network residual error to hidden layers. A hidden node itself can be formed by several hidden nodes.

  • Contributions:

    1. Faster than BP, SVM and other ELMs and compatible to regreesion and classification problems.
    2. The regularized parameter C do not affect the generalization performance of this method.
    3. This method with m hidden nodes (the desired output dimensionality) can achieve better training accuracy than the original ELM with a large number of hidden nodes.

в…Ў. Definitions and Basic-ELM

A. Notations and Definitions

  • рқҗ‘ : set of real numbers

  • {(рқҗұбөў,рқҗӯбөў)}бөўвӮҢвӮҒбҙә : N arbitrary distinct samples,

  • рқҗұбөў = [xбөўвӮҒ,xбөўвӮӮ,…,xбөўвӮҷ]бөҖ : n-dim input data, рқҗұбөўвҲҲ рқҗ‘вҒҝ is a column vector;

  • рқҗӯбөў : m-dim desired output data, рқҗӯбөўвҲҲ рқҗ‘бөҗ

  • рқҗұ : input data matrix, рқҗұ=[рқҗұвӮҒ,…рқҗұ_N], рқҗұбөўвҲҲ рқҗ‘вҒҝб•Ғбҙә

  • рқҗӯ : desired output data matrix, рқҗӯ=[рқҗӯвӮҒ,…рқҗӯ_N], рқҗӯвҲҲ рқҗ‘бөҗб•Ғбҙә

  • (^рқҗҡвӮҷ,^bвӮҷ) : parameters of the рқ‘ӣth subnetwork hidden node, ^рқҗҡвӮҷвҲҲ рқҗ‘вҒҝб•Ғбөҗ, ^bвӮҷвҲҲ рқҗ‘ (suppose the number of hidden nodes equals to the output dimension m, thus the mapping is from n to m.)

  • ^рқҗҡвӮҷ = [рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ], nГ—m weights matrix (for a n-dimension sample рқҗұбөў), рқҗҡвӮҷвӮҳвҲҲ рқҗ‘вҒҝ

  • рқҗһвӮҷ : residual error of current network output рқ‘“вӮҷ with рқ‘ӣ hidden nodes (for N samples), i.e., рқҗһвӮҷ=рқҗӯ-рқ‘“вӮҷ, рқҗһвӮҷвҲҲ рқҗ‘бөҗб•Ғбҙә.

  • рқҗҮ : output matrix of the hidden layer (of SLFN) for tarining set {(рқҗұбөў,рқҗӯбөў)}бөўвӮҢвӮҒбҙә, рқҗҮ = [h(рқҗұвӮҒ),…,h(рқҗұ_N)]бөҖ, рқҗҮвҲҲ рқҗ‘бҙәб•Ғбөҗ, рқҗҮ = g(рқҗұбөҖрқҗҡ+рқҗӣ)???

  • h(рқҗұ) : activation function. ELM feature mapping (or Huang’s transform)

  • рқҗҮбөў : the рқ‘–th hidden node output w.r.t. inputs, i.e., the рқ‘–th column of рқҗҮ

  • рқҗҲ : unit matrix

  • sum(рқҗһ) : the sum of all elements of the matrix рқҗһ

B. Basic-ELM

ELM is proposed for single-hidden-layer feedforward networks (SLFNs).

  • The output function of ELM with L hidden nodes for SLFNs is:

    $рқ‘“_L(рқҗұ)$ = вҲ‘бөўвӮҢвӮҒбҙё ОІбөўвӢ…h(рқҗҡбөўвӢ…рқҗұвұј + bбөў) = вҲ‘бөўвӮҢвӮҒбҙё рқҗҮбөўвӢ…рқӣғбөў, j=1,…,N.

    where h(вӢ…) denotes an activation function, (рқҗҡбөў, bбөў), рқҗҡбөўвҲҲ рқҗ‘вҒҝ, bбөўвҲҲ рқҗ‘, denotes the ith hidden node parameters, and рқӣғбөў is the ith output weight between the ith hidden node and the output nodes.

  • Based on Bartlett’s theory, ELM theory aims to reach not only the smallest training error, but also the smallest norm of output weights (Least square-least norm solution, where the regularization makes an invertible matrix, such that a special solution can be determined.):

    Minimize: вҖ–рқӣғвҖ–ВІ + CвӢ…вҖ–рқҗҮрқӣғ - рқҗӯвҖ–ВІ

    “then the generalization performance depends on the size of weights rather than the number of nodes.”

Lemma 1 (proved by Huang):
Given an SLFN with nonconstant piecewise continuous hidden nodes рқҗҮ(рқҗұ, рқҗҡ, b), then for any continuous target function рқ‘“ and any function sequence рқҗҮвӮҷКі(рқҗұ) = рқҗҮ(рқҗұ, рқҗҡвӮҷ, bвӮҷ) randomly generated based on any continuous sampling distribution,

lim$_{nвһқвҲһ}$ вҖ–рқ‘“ - (рқ‘“вӮҷвӮӢвӮҒ + рқҗҮвӮҷКівӢ…рқӣғвӮҷ)вҖ– = 0

holds with probabitliy 1 if:
рқӣғвӮҷ = вҹЁрқҗһвӮҷвӮӢвӮҒ, рқҗҮвӮҷКівҹ© / вҖ–рқҗҮвӮҷКівҖ–ВІ (the weight of the рқ‘ӣth node)

where

  • “вҹЁ , вҹ©” stands for “dot product” (Frobenius inner product) of two matrices and is a scalar.
  • n is the number of hidden nodes in the hidden layer.
  • рқҗһвӮҷвӮӢвӮҒ is the residual error of the last iteration, i.e., when there were n-1 hidden nodes.
  • рқҗҮвӮҷКі is the output matrix of the current hidden layer (activated but havn’t scaled by рқӣғ).

Intuitively, as the residual error reduces, the weight of the newer node gets smaller.

в…ў. Proposed ELM Method With Subnetwork Hidden Nodes

A. Structure of the Proposed Method

Motivations:

  1. Selecting an appropriate number of neurons can resort to optimization algorithms.
  2. The generalization performance depends on the size of the weights rather than the number of weights.

Inspiration:

“A hidden node itself can be a subnetwork formed by several nodes. And these subnetwork hidden nodes and output weights itself should be the smallest norm, and also aim to reach the smallest training error.”

Objectives:

Given N training samples {(рқҗұбөў,рқҗӯбөў)}бөўвӮҢвӮҒбҙә, рқҗұбөўвҲҲ рқҗ‘вҒҝ, рқҗӯбөўвҲҲ рқҗ‘бөҗ, generated from the same continuous system, if activation function h is invertible, the objectives are:

  1. вҖ–рқҗһвӮҷвӮӢвӮҒвҖ– вүҘ вҖ–рқҗһвӮҷвӮӢвӮҒ - h(^рқҗҡвӮҷ,рқҗұ)вҖ– вүҘ вҖ–рқҗһвӮҷвӮӢвӮҒ - рқҗҮвӮҷвҖ– вүҘ вҖ–рқҗһвӮҷвӮӢвӮҒ - рқҗҮвӮҷвӢ…рқӣғвӮҷвҖ– (residual error is decreasing.)
  2. вҖ–h(^рқҗҡвӮҷ,рқҗұ) - рқҗһвӮҷвӮӢвӮҒвҖ– = min_{рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ} вҖ–h(рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ) - рқҗһвӮҷвӮӢвӮҒвҖ– (minimize weights inside nodes)
  3. вҖ–рқҗҮвӮҷвӢ…^рқӣғвӮҷ - рқҗһвӮҷвӮӢвӮҒвҖ– = min_{рқӣғ} вҖ–рқҗҮвӮҷвӢ…рқӣғ - рқҗһвӮҷвӮӢвӮҒвҖ– (minimize the weights outside nodes)

where

  • ^рқҗҡвӮҷ and ^рқӣғвӮҷ are the optimal (the ultimate status) parameters with the smallest norm among all the least squares solutions.
  • рқҗҮвӮҷ = h(^рқҗҡвӮҷ, ^bвӮҷ, рқҗұ) is the output of the nth hidden node with the optimal parameters.

If activation function h is invertible, subnetwork hidden nodes in SLFN can be calculated by pulling back network residual error to hidden layers.

  • For example, with sine function as the activation function, training a subnetwork hidden node (^рқҗҡ) is equivalent to finding a least-square solution ^рқҗҡвӮҷ (letting the derivative of MSE=0) with the least norm for the linear system:

    [рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ]вӢ…рқҗұ = arcsin(рқҗһвӮҷвӮӢвӮҒ), рқҗһвӮҷвӮӢвӮҒвҲҲ (0,1],

    such that the optimal ^рқҗҡвӮҷ satifies:

    вҖ–sin(^рқҗҡвӮҷ, рқҗұ) - рқҗһвӮҷвӮӢвӮҒвҖ– = min_{рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ} вҖ–sin(рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ, рқҗұ) - рқҗһвӮҷвӮӢвӮҒвҖ–,

    That means the output of the nth “subnetwork hidden node” ^рқҗҡвӮҷ is approaching the residual error рқҗһвӮҷвӮӢвӮҒ of the last status.

    The input weights ^рқҗҡвӮҷ of a node for this model is a matrix (instead of a vector), beacuse each “subnetwork (general) hidden node” contains a standard SLFN (several hidden nodes) internally.

B. Proposed Method

Lemma 2:
Given a bounded nonconstant piecewise continuous activation function h, there is:
lim$_{(рқҗҡ,b)вҶ’(рқҗҡвӮҖ,bвӮҖ)}$ вҖ–h(рқҗҡвӢ…рқҗұ+b) - h(рқҗҡвӮҖвӢ…рқҗұ+bвӮҖ)вҖ– = 0 (иҝһз»ӯжҖ§)

Theorem 1:
Given N arbitrary distinct samples {(рқҗұбөў,рқҗӯбөў)}бөўвӮҢвӮҒбҙә, рқҗұбөўвҲҲ рқҗ‘вҒҝ, рқҗӯбөўвҲҲ рқҗ‘бөҗ, a sigmoid or sine activation function h, and then for any continuous desired outputs рқҗӯ, the limit of error converges to 0:

lim$_{nвһқвҲһ}$ вҖ– рқҗӯ-{ uвҒ»В№(h(^рқҗҡвӢ…рқҗұ+b))} вҖ– = 0

Proof:

  • Prove the sequence вҖ–рқҗһвӮҷвҖ– is decreasing with 0 as the lower bound and it converges.

    1. For the рқ‘ӣth subnetwork hidden node containing m hidden nodes, the linear mapping is:

      рқӣҢвӮҷ = [рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ]вӢ…рқҗұ, рқӣҢвӮҷвҲҲ рқҗ‘бөҗ

    2. Then рқӣҢвӮҷ passes through the activation function. Because the target is error, which should become 0 at the end, the error at present is the output of activation function:

      рқҗһвӮҷвӮӢвӮҒ = h(рқӣҢвӮҷ) вҲҲ рқҗ‘бөҗ

    3. The inverse function of h is hвҒ»В№, and its input value should range from (0,1]. Therefore, if trying to solve рқӣҢвӮҷ from рқҗһвӮҷвӮӢвӮҒ, every element in рқҗһвӮҷвӮӢвӮҒ should be scaled to the range of (0,1] by the normalized function u(вӢ…). Then, рқӣҢвӮҷ can be calculated through:

      рқӣҢвӮҷ = hвҒ»В№(u(рқҗһвӮҷвӮӢвӮҒ))

    4. Further, the input weights of this subnetwork hidden node can be solved:

      $$\^рқҗҡвӮҷ = [рқҗҡвӮҷвӮҒ,…,рқҗҡвӮҷвӮҳ] = hвҒ»В№(u(рқҗһвӮҷвӮӢвӮҒ))вӢ…рқҗұвҒ»В№$$

    5. For different activation functions, there will be:

      $$\rm \{^{\^рқҗҡвӮҷ = arcsin(u(рқҗһвӮҷвӮӢвӮҒ))вӢ…рқҗұвҒ»В№,\quad sine} _{\^рқҗҡвӮҷ = -log( (1/u(рқҗһвӮҷвӮӢвӮҒ)) - 1)вӢ…рқҗұвҒ»В№,\quad sigmoid}$$

    (This work is a continuation on ELM, that is once the рқӣғ calculated based on target рқҗӯ and рқҗҮвҒ»В№, the residual error is also fixed, so it serves as the target for рқҗҡ,b. Still, applying least-square, the optimial a can be calculated based on “target” e and рқҗұвҒ»В№.)

    1. ^b is the mean of hidden nodes.

    2. The error can be reduced by adding the bias

    3. Do feedforward using the calculated ^рқҗҡвӮҷ and ^bвӮҷ, so this time the output of the hidden layer is:

      $$\^рқҗҮвӮҷбөү = uвҒ»В№(h(\^рқҗҡвӮҷвӢ…рқҗұ+\^bвӮҷ))$$

      Because рқҗһвӮҷвӮӢвӮҒ is the last output of the activation fuction, рқҗһвӮҷвӮӢвӮҒ and $\^рқҗҮвӮҷбөү$ are the same things. So they can subtract from each other. Then the residual error for this time is:

      О” = вҖ–рқҗһвӮҷвӮӢвӮҒвҖ–ВІ - вҖ–рқҗһвӮҷвӮӢвӮҒ - ^рқҗҮвӮҷбөүвҖ–ВІ
      = вҖ–рқҗһвӮҷвӮӢвӮҒвҖ–ВІ - (вҖ–рқҗһвӮҷвӮӢвӮҒвҖ–ВІ - 2вҖ–рқҗһвӮҷвӮӢвӮҒвҖ–вҖ–^рқҗҮвӮҷбөүвҖ– + вҖ–^рқҗҮвӮҷбөүвҖ–ВІ)
      = 2вҖ–рқҗһвӮҷвӮӢвӮҒвҖ–вҖ–^рқҗҮвӮҷбөүвҖ– - вҖ–^рқҗҮвӮҷбөүвҖ–ВІ
      = 2вҹЁрқҗһвӮҷвӮӢвӮҒ, ^рқҗҮвӮҷбөүвҹ© - вҖ–^рқҗҮвӮҷбөүвҖ–ВІ
      = вҖ–^рқҗҮвӮҷбөүвҖ–ВІ ( 2вҹЁрқҗһвӮҷвӮӢвӮҒ, ^рқҗҮвӮҷбөүвҹ©/вҖ–^рқҗҮвӮҷбөүвҖ–ВІ - 1 )

    4. О” is вүҘ 0

  • Prove the limit converges to 0 when n tends to infinity.

The target value is approximated while the error is decreased.

The final estimation is the summation of the ouput of d subnetwork hidden nodes

The VC dimension is lower than standard ELM, i.e., the dimension of feature space mвүӘ L, so the generalization ability of this method is better.
