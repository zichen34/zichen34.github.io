DDG search: “point cloud fusion”, found the wikipedia page of “Point cloud”, where the caption of the illustration 3 mentioned: “reconstructing 3D shapes from multi-view depth map”. That picture is from:

  1. Synthesizing 3D Shapes via Modeling Multi-View Depth Maps and Silhouettes with Deep Generative Networks ~ CVPR 2017

    Code | Amir A. Soltani, Jiajun Wu

Then I searched the paper’s title in DDG , and I found:

  1. Render4Completion: Synthesizing Multi-View Depth Maps for 3D Shape Completion ~ ICCV 2019 workshop

    Arxiv | CoolPapers | Tao Hu, Zhizhong Han, Matthias Zwicker

    • Data: Render multi-view depth maps for incomplete object

    • Train: 8-view incomplete depth map -> NN -> 8-view completed depth map

    • 3D shape: Unproject 8 depth maps

  1. Learning Implicit Fields for Generative Shape Modeling ~ CVPR 2019

    Arxiv | Zhiqin Chen, Hao Zhang, SFU

  1. Improved Modeling of 3D Shapes with Multi-view Depth Maps ~ 3DV 2020

    Code | CoolPapers | Kamal Gupta, Matthias Zwicker
