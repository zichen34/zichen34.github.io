Notes:

(2024-10-28)

  1. Nano’s pin-13 is the built-in on-board LED (r1-Docs).

  2. Pins of Nano:

    (2024-10-29)

    1. A stands for Analog;

    2. D stands for Digital;

    3. The bi-directional pins can be set as INPUT or OUTPUT modes;

    4. VIN and Micro USB port CANNOT simultaneously plug in, as they both are input source.

      • VIN should be input 7 - 12 V voltage.

    5. RX and TX can be used as INPUT/OUTPUT pins, as long the Serial library is not called.
